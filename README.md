# Convergent Systems Scoop Bucket

A [Scoop](https://scoop.sh) bucket for Convergent Systems CLI tools.

## Install

```powershell
scoop bucket add convergent-systems https://github.com/convergent-systems-co/scoop-bucket
```

## Available packages

| Package | Description | Source |
|---|---|---|
| `gitignore` | A free CLI for managing `.gitignore` files | [convergent-systems-co/gitignore](https://github.com/convergent-systems-co/gitignore) |

## Usage

```powershell
# After adding the bucket:
scoop install gitignore
scoop update gitignore
scoop uninstall gitignore
```

Manifests in this bucket are updated automatically via GitHub Actions when their upstream repos cut a release.
